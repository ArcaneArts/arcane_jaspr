# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.1] - 2024-12-17

### Added
- Initial release of arcane_jaspr
- Core theming system (`ArcaneTheme`, `ContrastedColorScheme`)
- Layout components: `Section`, `Gutter`, `Flow`, `Carpet`, `ButtonPanel`, `RadioCards`
- Input components: `ArcaneButton`, `ArcaneIconButton`, `CycleButton`, `Search`, `Selector`
- View components: `Bar`, `Glass`, `ArcaneCard`, `CardSection`, `Tile`, `Expander`, `CenterBody`
- Navigation components: `Sidebar`, `BottomNavigationBar`, `ArcaneTabs`
- Screen components: `Screen`, `FillScreen`, `NavigationScreen`
- Dialog components: `ArcaneDialog`, `ConfirmDialog`, `TextInputDialog`, `Toast`
- Form components: `Field`, `FieldWrapper`, `FormProvider`
- Collection components: `Collection`, `CardCarousel`
- Table components: `DataTable`, `StaticTable`
- Utility classes and extensions for web development
